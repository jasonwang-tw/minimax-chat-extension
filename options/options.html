API 設定

用於一般對話與文生圖，請至 MiniMax Token Plan 取得。

用於截圖與圖片分析，免費申請 Google AI Studio。

免費申請 Brave Search API。有此 Key 時優先使用。

免費申請 Exa Search API。無 Brave Key 時作為備援。

對話設定

設定預設模型後，側欄聊天會優先使用此選擇。

提示詞

你可以在這裡設定全域與各工具的系統提示詞。

回覆模式

自訂不同的回覆模式，讓聊天切換更快。

自訂指令

使用 `/指令` 的方式快速插入模板或觸發行為，模板可使用 {input} 佔位。

長期記憶

AI 可以根據對話自動萃取值得保存的內容，也可以用 /remember 手動新增。

開啟後，每次對話結束會額外執行一次記憶萃取。

跨裝置同步（Phase 1）

目前以 WordPress 備援同步為主，支援手動/自動備份與還原。備份範圍包含設定、長期記憶、知識庫、單字簿與 Session 內容。

啟用後會在指定時間自動執行 WordPress 備份。
